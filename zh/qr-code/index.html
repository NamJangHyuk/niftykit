什么是二维码生成器？

二维码生成器是一款可以将网址、Wi-Fi 信息、简短消息等任意文本立即转换为二维码的工具。无论是名片、菜单、海报还是演示文稿都需要二维码时，无需安装任何软件即可立即生成。生成的二维码可免费下载为 PNG 或 SVG 文件，所有处理均在浏览器中即时完成。

使用方法

首先选择您需要的二维码类型（网址、文本、Wi-Fi、联系人、邮件、电话、短信、日程或位置），然后填写相应字段，输入的同时下方会立即显示二维码预览。满意后点击「PNG」或「SVG」按钮即可下载文件。如果输入内容过长无法生成二维码，页面会显示提示信息。

适用场景

  • 快速分享 Wi-Fi 密码或网站地址
  • 在名片或宣传单上添加二维码
  • 将印刷菜单链接到在线点餐页面
  • 在演示文稿或海报中嵌入二维码

常见问题

可以免费使用吗？
可以，无需注册即可免费生成并下载二维码。
可以下载哪些格式？
支持PNG和SVG两种格式。PNG适合直接用于文档和打印，SVG是放大也不会失真的矢量格式。
输入的字数有限制吗？
由于二维码本身的容量限制，内容过长可能无法生成。若内容过长，页面会显示提示信息，建议使用较短的网址或精简文本。
可以制作Wi-Fi或名片二维码吗？
可以。在顶部分类中选择Wi-Fi、联系人、邮件、电话、短信、日程或位置，即可生成对应用途的二维码。例如Wi-Fi二维码在扫描后会自动连接到该网络。

本工具的二维码生成过程完全在您的浏览器本地完成，您输入的内容不会被发送或存储到服务器。
