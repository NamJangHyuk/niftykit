유저네임 생성기

영단어를 무작위로 조합해서 새로운 아이디 아이디어를 얻어보세요

구분자, 대소문자 스타일, 숫자 추가 여부를 원하는 대로 조정할 수 있습니다.

이메일 아이디·유저네임에 바로 쓸 수 있도록, 결과는 화면 언어와 상관없이 항상 영문으로 생성됩니다.

단어 목록을 불러오지 못했습니다. 페이지를 새로고침해 주세요.

유저네임 생성기란?

유저네임 생성기는 이메일 주소나 아이디, 유저네임처럼 영문으로 지어야 하는 이름을 정할 때 쓰는 도구입니다. 형용사 2개와 명사 1개(옵션에 따라 조합 방식 변경 가능)를 무작위로 조합해 새로운 이름을 즉시 만들어주며, 새 이메일 주소를 만들 때, 게임이나 온라인 커뮤니티에서 쓸 닉네임이 필요할 때, 프로젝트나 저장소에 붙일 코드네임이 필요할 때 흔히 겪는 "이름을 못 정하겠다"는 문제를 바로 해결해줍니다. 수백 개씩 큐레이션한 형용사·명사 단어 뱅크에서 브라우저의 암호학적으로 안전한 난수 생성기(crypto.getRandomValues)로 골라 조합하기 때문에, 매번 진짜로 예측 불가능한 조합이 나옵니다. 화면 언어와 관계없이 생성되는 결과는 실제 이메일·아이디로 바로 쓸 수 있도록 항상 영문으로만 나옵니다.

사용 방법

"다시 생성" 버튼을 누르면 바로 무작위 조합이 나옵니다. 구분자 드롭다운에서 점·언더스코어·하이픈 중 원하는 걸 고르거나 아예 붙여쓸 수도 있고, 가입하려는 서비스 성격에 맞게 대소문자 스타일(소문자/camelCase/PascalCase)도 바꿀 수 있습니다. "숫자 추가"를 켜면 뒤에 무작위 숫자가 붙는데, 단어 조합만으로는 이미 사용 중인 경우가 많아서 켜두는 걸 추천합니다. 마음에 드는 결과가 나오면 "복사" 버튼으로 클립보드에 바로 복사하세요.

이럴 때 유용해요

  • 새 이메일 주소의 아이디를 정할 때
  • 게임이나 온라인 커뮤니티에서 쓸 닉네임이 필요할 때
  • 새 프로젝트, 저장소, 코드네임을 지을 때
  • 원하던 아이디가 이미 사용 중이어서 아이디어가 필요할 때

자주 묻는 질문

단어는 어떻게 조합되나요?
서로 다른 형용사 2개와 명사 1개를, 각각 수백 개씩 있는 큐레이션된 단어 뱅크에서 무작위로 골라 조합합니다.
생성된 결과가 서버로 전송되나요?
아니요. 모든 조합은 Web Crypto API를 이용해 브라우저 안에서만 생성되며, 서버로 전송되거나 저장되지 않습니다.
숫자 없이 아이디를 받을 수 있나요?
네. "숫자 추가" 체크박스를 끄면 선택한 구분자로만 연결된 세 단어만 나옵니다.
화면이 한국어인데 결과도 한국어로 나오나요?
아니요. 이 도구는 이메일 아이디·유저네임 네이밍이 목적이라, 화면 언어와 상관없이 결과는 항상 영문으로만 생성됩니다.
Math.random() 대신 crypto.getRandomValues()를 쓰는 이유는 무엇인가요?
Math.random()은 예측 가능한 의사난수라 진짜 무작위성이 중요한 상황에는 적합하지 않습니다. crypto.getRandomValues()는 브라우저가 보안이 중요한 작업에 쓰는 것과 동일한 암호학적으로 안전한 난수 소스라, 조합이 특정 패턴으로 예측되지 않습니다.

이 도구는 모든 유저네임 생성을 브라우저 안에서만 처리하며, 생성된 결과를 서버로 전송하거나 저장하지 않습니다.
