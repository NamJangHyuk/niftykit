Password Generator

대문자·숫자·특수문자를 조합해 안전한 비밀번호를 무료로 만들어드려요

길이와 문자 종류를 자유롭게 설정하고, 비밀번호 강도까지 바로 확인하세요.

문자 종류를 하나 이상 선택하세요

비밀번호 생성기란?

비밀번호 생성기(Password Generator)는 대문자, 소문자, 숫자, 특수문자를 원하는 대로 조합해 안전한 랜덤 비밀번호를 즉시 만들어주는 도구입니다. 브라우저의 암호학적으로 안전한 난수 생성 기능(crypto.getRandomValues)을 사용하기 때문에 예측이 거의 불가능한 진짜 무작위 비밀번호를 얻을 수 있습니다. 새 계정을 만들 때나 기존 비밀번호를 더 안전한 것으로 바꾸고 싶을 때 바로 사용할 수 있습니다.

사용 방법

원하는 비밀번호 길이를 슬라이더로 정하고, 포함할 문자 종류(대문자·소문자·숫자·특수문자)를 체크하세요. 옵션을 바꿀 때마다 비밀번호가 자동으로 새로 만들어지고, 화면에는 강도(약함~매우 강함)도 함께 표시됩니다. 마음에 드는 비밀번호가 나오면 "복사" 버튼을 눌러 클립보드에 바로 복사하세요.

이럴 때 유용해요

  • 새로운 계정을 만들 때 안전한 비밀번호가 필요할 때
  • 기존 비밀번호를 더 강력한 것으로 바꾸고 싶을 때
  • 여러 사이트마다 서로 다른 비밀번호를 쓰고 싶을 때
  • Wi-Fi 공유기나 기기의 새 비밀번호를 정할 때

자주 묻는 질문

생성된 비밀번호가 안전한가요?
네. Math.random() 대신 브라우저의 암호학적 난수 생성 기능(crypto.getRandomValues)을 사용해서 예측이 거의 불가능한 진짜 무작위 값으로 만듭니다.
비밀번호가 서버에 저장되나요?
아니요. 모든 생성 과정은 브라우저 안에서만 이루어지며, 입력하거나 생성된 어떤 값도 서버로 전송되거나 저장되지 않습니다.
비밀번호 길이는 어느 정도가 안전한가요?
일반적으로 12자 이상을 권장하며, 중요한 계정일수록 16자 이상에 대문자·소문자·숫자·특수문자를 모두 포함하는 것이 안전합니다.

이 도구는 비밀번호 생성을 브라우저 로컬에서만 처리하며, 생성된 비밀번호를 서버로 전송하거나 저장하지 않습니다.
