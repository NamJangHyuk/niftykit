White Noise

화이트·핑크·브라운 노이즈를 그 자리에서 합성해 들려드려요

별도 음원 파일 다운로드 없이 브라우저가 직접 소리를 만들어내며, 볼륨과 자동 종료 시간을 조절할 수 있어요.

정지됨

노이즈 종류

백색소음 플레이어란?

백색소음 플레이어(White Noise)는 화이트, 핑크, 브라운 세 가지 노이즈를 브라우저의 Web Audio API로 그 자리에서 직접 합성해 들려주는 도구입니다. 실제 자연음을 녹음한 파일을 다운로드하는 방식이 아니라 코드로 소리 파형을 만들어내기 때문에 저작권 문제가 전혀 없고, 인터넷 연결이 느려도 끊김 없이 재생됩니다. 화이트 노이즈는 모든 주파수가 균일하게 섞인 소리이고, 핑크 노이즈는 저음이 더 강조되어 부드럽게 들리며, 브라운 노이즈는 저음이 한층 더 강조되어 파도 소리처럼 묵직하게 들립니다.

사용 방법

원하는 노이즈 종류를 선택하고 "재생" 버튼을 누르면 바로 소리가 나옵니다. 재생 중에도 볼륨 슬라이더로 크기를 조절할 수 있고, 자동 종료 시간을 미리 설정해두면 그 시간이 지난 뒤 저절로 재생이 멈춥니다. 노이즈 종류를 바꾸려면 먼저 "정지" 버튼을 눌러 재생을 멈춘 뒤 다시 선택하면 됩니다.

이럴 때 유용해요

  • 공부나 업무 중 주변 소음을 덮어 집중력을 유지하고 싶을 때
  • 잠들기 전 편안한 배경음이 필요할 때(자동 종료 타이머로 밤새 재생 방지)
  • 사무실이나 카페의 소음을 차단하고 싶을 때
  • 아기를 재우거나 명상할 때 일정한 배경음이 필요할 때

자주 묻는 질문

노이즈 종류마다 어떤 차이가 있나요?
화이트 노이즈는 모든 주파수가 고르게 섞인 소리로 가장 또렷하고, 핑크 노이즈는 저음이 강조되어 조금 더 부드럽게 들리며, 브라운 노이즈는 저음이 훨씬 더 강조되어 파도나 폭포 소리처럼 묵직하게 들립니다.
실제 자연음 파일을 다운로드하는 건가요?
아니요. 이 도구는 음원 파일을 전혀 사용하지 않고, 브라우저에서 Web Audio API로 소리 파형을 실시간으로 계산해서 만들어냅니다. 그래서 저작권 문제가 없고 용량도 필요 없습니다.
자동 종료 타이머는 어떻게 동작하나요?
재생을 시작하기 전에 자동 종료 시간을 선택해두면, 그 시간이 지난 순간 재생이 자동으로 멈춥니다. "끄기"로 두면 직접 정지 버튼을 누를 때까지 계속 재생됩니다.

이 도구는 소리 합성과 재생을 브라우저에서만 처리하며, 서버로 전송하거나 저장하는 데이터가 없습니다.
